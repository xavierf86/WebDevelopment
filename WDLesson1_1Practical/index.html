Web Technologies

HTML

HTML stands for Hypertext Markup Language, and its a coding language that displays content on a webpage.
HTML is not a programming language. Gives the webpages structure onadding fonts, images, videos, forms and more.

CSS

CSS stands for Cascading Styles Sheets, and it describes how HTML elements are displayed on the screen, paper, or in other medias.
Basically styles your layout and displays life to the HTML screen with colors.

JavaScript

JavaScript is a programming language that is used to enable interactive and dynamic content on webpages.
Makes it so that you can implement complex features on web pages.

JQuery

JQuery is a JavaScript library, that basically simplifies HTML DOM tree traversal and manipulation, event handling, and CSS animations.
It makes client side scripting easier by providing a simple and consistent API across different web browsers.
